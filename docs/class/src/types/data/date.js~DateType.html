    DateType

    Extends:

    Type → DataType → DateType

    The DataType for date values. Stores a value of seconds since January 1st 1970.

    Allowed options:

    Name Type Attribute Description
    dateType string
    • optional
    • default: 'datetime'
    		 Whether to handle just a date, time or both. 'date' | 'time' | 'datetime'
